# Logo

Placez ici le logo de l'application Allô Dakar.

## Formats recommandés
- PNG avec transparence
- SVG (si possible)
- Formats : `.png`, `.jpg`, `.svg`

## Tailles recommandées
- Logo principal : 512x512px ou 1024x1024px
- Icône app : 1024x1024px (pour les stores)
- Logo petit : 256x256px (pour les headers)

## Utilisation dans le code

```typescript
import logo from '../../assets/images/logo/logo.png';

// Dans un composant
<Image source={logo} style={styles.logo} />
```
